# Бахш

Метод / Хосият

Вазифаи асосӣ

Сатрҳо (Ҷустуҷӯ)

.length

Фаҳмидани дарозии сатр (шумораи аломатҳо)



.includes(str)

Санҷиши мавҷудияти зерсатр дар даруни сатр (true/false)



.indexOf(str)

Ёфтани индекси аввалин вохӯрии зерсатр (ё -1 агар наёбад)



.startsWith(str) / .endsWith(str)

Санҷиши он ки сатр бо кадом аломатҳо оғоз меёбад ё анҷом мепазирад

Сатрҳо (Ҷарроҳӣ / Буридан)

.slice(start, end)

Буридани қисми матн ба таври дақиқ (усули тавсияшаванда)



.substring(start, end)

Аналоги камтар сахтгири slice бо ивазкунии автоматии индексҳо

Сатрҳо (Таҳрир ва Тозакунӣ)

.trim()

Дур кардани фосилаҳо (пробелҳо) аз аввал ва охири сатр



.toLowerCase() / .toUpperCase()

Тағйир додани регистри аломатҳо (ҳарфҳои хурд / калон)



.replace() / .replaceAll()

Иваз кардани зерсатр(ҳо) бо сатри нав



.split(separator)

Тақсим кардани сатр ба массиви элементҳо аз рӯи разделител

Ададҳо ва Math

Number(str) / +str

Табдилдиҳии бехатари сатр ба адад



isNaN(val)

Санҷиши он ки оё қимат NaN (қимати нодурусти ададӣ) аст ё не



Math.floor(x)

Гирд кардани адад ба самти поён («фарш»)



Math.ceil(x)

Гирд кардани адад ба самти боло («шифт»)



Math.round(x)

Гирд кардан тибқи қоидаҳои стандартии математика (ба адади наздиктарин)



Math.abs(x)

Мусбат кардани адад (модули адад)



Math.max(...) / Math.min(...)

Ёфтани адади калонтарин / хурдтарин аз байни аргументҳо



Math.random()

Гирифтани адади тасодуфии касрӣ аз 0 (бо назардошт) то 1 (бе назардошт)
